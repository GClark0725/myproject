HOW TO MAKE A WEBSITE

The Languages

To make a website, you will need to know a couple of languages:
  • html
  • css
  • javascript
  • sinatra

If you know little to nothing about the above codes, go to


The Files

Now that you understnd the code, you need to write the code on seperate files.
Once you do that, you will want to make a new blank folder. Then you will make the following folders inside that folder:
  • views
  • public
Then, you will put your html file(s) in the views folder and your css file(s) and any pictures you have in your public folder.
Your javascript files will go in the folder that contains the views and public folders.


Running the File

To run the file, you will go to your terminal and find the javascript file.
Run the javascript file, and then go to your browser and type localhost4567
